About the Project

This project was inspired by a personal desire to create an inclusive and accessible learning environment for children with Down syndrome. Witnessing the challenges these children face in traditional learning settings, I wanted to build a platform that caters specifically to their needs. The journey began in early 2023, and after months of development and testing, this platform has evolved into a comprehensive tool that aims to enhance learning experiences for these children.

This project is part of my portfolio for Holberton School, showcasing my skills in web development and design. You can explore the code and more details in the Github Repository.
